# Hyperskill Projects — Ahmad

A collection of project-based learning work built while studying on [Hyperskill](https://hyperskill.org){:target="_blank"}, JetBrains' project-based platform
for software development skills.

## About this organization

Each repository here represents a complete, independently-built project
based on a Hyperskill project brief. Rather than isolated exercises,
these are small applications built stage by stage — covering real-world
concerns like file handling, APIs, data processing, and application
structure.

## Focus

- **Currently:** Python
- **Planned:** additional tracks/languages as they're added

## How these projects are built

- Solved independently based on the public Hyperskill project
  descriptions
- Each repository includes its own README with setup instructions and
  project status

## Purpose

This organization was created to record my personal learning path —
it's primarily for myself, as a way to track progress and look back
on how each project was approached over time.
